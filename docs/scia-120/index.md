## Course Overview

Broad introduction to computing security and information assurance. Students learn to think like security professionals — understanding what they are defending, who threatens it, and what tools and practices protect it. Topics span the full landscape of modern information security, from physical controls and social engineering through cryptography, network defense, secure programming, and cloud security.

!!! abstract "What You Will Learn"
    By the end of this course, you will be able to:

    - Define information security and explain the CIA Triad (Confidentiality, Integrity, Availability)
    - Identify physical security threats and apply appropriate countermeasures
    - Recognize and defend against social engineering attacks
    - Harden operating systems and manage user access controls
    - Identify and analyze common malware types and their behaviors
    - Apply cryptographic principles including hashing, symmetric, and asymmetric encryption
    - Understand network security architecture, firewalls, and intrusion detection
    - Identify web application vulnerabilities (OWASP Top 10) and secure coding practices
    - Evaluate authentication mechanisms and access control models
    - Apply risk management frameworks and security policy principles
