# VS Code Remote Development using SSH

Accessed: 2026-04-28

This source supports the VS Code Remote SSH setup page.

Key classroom-relevant points:

- VS Code Remote - SSH lets students open folders on an SSH-accessible server and work with a near-local editing experience.
- The remote machine needs a running SSH server.
- The local machine needs VS Code, an OpenSSH-compatible SSH client, and the Remote - SSH extension.
- Key-based SSH authentication is recommended.
- VS Code installs a VS Code Server component on the remote host during first connection.
- Students should verify terminal SSH first before troubleshooting VS Code.
