# DigitalOcean How To Install Git on Ubuntu

Accessed: 2026-04-28

This source supports the Git and GitHub setup page.

Key classroom-relevant points:

- Git can be installed from Ubuntu's default APT repositories for a stable, straightforward setup.
- Verify installation with `git --version`.
- Configure `user.name` and `user.email` so commits are properly attributed.
- Review global Git configuration with `git config --global --list`.
- Use `main` as a modern default branch name.
