## Course Overview

This course examines the principles and practices of securing data and information within database environments. Students explore relational database management system (RDBMS) architecture through a security lens, learning how databases can be compromised and how to defend them. Topics include SQL security and injection prevention, database authentication and hardening, role-based access control, encryption of data at rest and in transit, network-level database security, and comprehensive auditing strategies. The course concludes with cloud database security, NoSQL considerations, compliance frameworks, and incident response procedures for database breaches.

!!! abstract "What You Will Learn"
    By the end of this course, you will be able to:

    - Describe the security model of relational database management systems and identify common attack surfaces
    - Identify and mitigate SQL injection vulnerabilities and other SQL-based attack vectors
    - Implement database authentication, authorization, and role-based access control policies
    - Apply encryption techniques to protect data at rest and in transit within database systems
    - Configure database auditing and logging to detect and investigate unauthorized activity
    - Conduct vulnerability assessments and penetration tests against database environments
    - Apply security principles to cloud-hosted and NoSQL database platforms
    - Align database security practices with compliance and regulatory standards such as PCI-DSS, HIPAA, and GDPR
    - Perform incident response and forensic analysis following a database security event
